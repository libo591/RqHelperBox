润乾报表在加密这方面的措施有很多，例如加密狗，或给赋予权限的用户开放浏览权限，但这些都是在设计阶段采取的措施，或者说一旦报表在WEB端展现了，客户还是可以通过复制页面内容等一些投机的手段在其他环境下查看报表中的数据。

本文简单介绍如何禁止页面复制来达到保密的效果。

一：禁止页面内容复制。

在jsp的<body>标签中加入oncontextmenu=”return false” onselectstart=”return false” ondragstart=”return false”这三个标签的含义分别是：禁止鼠标右键弹出IE菜单，禁止鼠标选取，禁止鼠标拖拽。

<body topmargin=0 leftmargin=0 rightmargin=0 bottomMargin=0 oncontextmenu=”return false” onselectstart=”return false” ondragstart=”return false”>

</body>

二：禁止右键

<SCRIPT language=javascript>

function click() {

alert(’禁止你的左键复制！‘) }

function click1() {

if (event.button==2) {alert(’禁止右键点击！‘) }}

function CtrlKeyDown(){

if (event.ctrlKey) {alert(’禁止复制！‘) }}

document.onkeydown=CtrlKeyDown;

document.onselectstart=click;

document.onmousedown=click1;

</SCRIPT>

三：禁止页面另存为。

在jsp的<body>标签加入以下代码：

<noscript>

<iframe src=”*.htm”></iframe>

</noscript>

将上面的方法加入jsp就可以控制页面只能看，不能复制转载等操作了，在一定程度上解决了网页的保密性。
