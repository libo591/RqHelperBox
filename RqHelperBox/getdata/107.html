问题

使用润乾报表的过程中发现在加载授权文件时“报表证书错误“

操作系统 Red Hat AS 5 (update 1)

使用weblogic8.1 sp5英文版自带的j2sdk- 1.4.2 _08

报错信息:

[2007-12-26 17:46:01] runqianReportLogger : [INFO ] - 报表证书错误!

/weblogic/bea/jdk142_08/jre/lib/i386/libawt.so: libXp.so.6: cannot

open shared object file: No such file or directory

java.lang.UnsatisfiedLinkError: /weblogic/bea/jdk142_08/jre/lib/i386/

libawt.so: libXp.so.6: cannot open shared object file: No such file or directory

at java.lang.ClassLoader$NativeLibrary.load(Native Method)

问题解决:

java图片处理需要本地X86动态连接库支持,缺少支持打印的图形化动态链接库libXp.so.6,

下载并安装XFree86-libs- 4.2.0 -8.i386.rpm，

/usr/X11R6/lib下就会出现需要的libXp.so.6.2
