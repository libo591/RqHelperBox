java 应用有不少是 C/S的模式，在 C/S的模式下，同样可以调用润乾的 API 接口运算报表，并且把结果报表在 JFrame 窗口中进行展现，还可以进行打印。

代码示例：
        IReport iReport = ……; //运算代码，获得一个 IReport 对象
        CSReport csReport = new CSReport(iReport); //构造CSReport控件
        JFrame jf = new JFrame(“润乾报表”); //构造 JFrame 界面
        jf.getContentPane().add(csReport.getDisplayPane()); //设置 JFrame 的相关属性并展现
        jf.setSize(600, 400);
        jf.setLocation(200, 100);
        jf.show();
        //……………. //其它代码
        //csReport.print(); //打印，该代码可以写在某个按钮的相关事件中
