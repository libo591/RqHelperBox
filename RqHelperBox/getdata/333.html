一、问题描述

超链接转到另一张报表之后，再想后退怎么办呢？

二、解决方法

用IE的后退或自己写后退前进按钮。

<script language=”javascript”>
       function goHist(a)
       { history.go(a); }
</script>
<form method=”post”>
       <input type=”button” value=”后退” onClick=”goHist(-1)” name=”button”>
       <input type=”button” value=”前进” onClick=”goHist(1)” name=”button”>
</form>
