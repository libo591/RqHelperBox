有些客户在设计报表的时候希望在设计器启动的时候就直接启动demo应用，以便设计的时候不用再去点击启动tomcat 的按钮，有一些客户则直接使用demo应用作为正式的应用，希望启动设计器的时候直接启动demo应用，这些需求的结果是一样的，但是这个如何满足呢，

进入设计器安装根目录的bin目录

看一下两个bat文件，

 

将这startdemo.bat文件里的内容

echo on

call setEnv.bat

start “database” %EXECJAVA%  -cp %RunqianCLASSPATH% -Dstart.home=%START_HOME% com.runqian.starter.StartUtil com.runqian.starter.EmbedeDataBaseProcessor start

start “runqianreport webserver” %EXECJAVA% -cp %RunqianCLASSPATH% -Dstart.home=%START_HOME% com.runqian.starter.StartUtil com.runqian.starter.EmbededTomcatProcessor start demo

追加到startup.bat文件里，成为这样:

@echo off

call setEnv.bat

start “runqianreport” %EXECJAVAW% -Xms64m -Xmx128m  -cp %RunqianCLASSPATH% -Dstart.home=%START_HOME% com.runqian.starter.StartUtil com.runqian.starter.Report4DesignerProcessor

echo on

call setEnv.bat

start “database” %EXECJAVA%  -cp %RunqianCLASSPATH% -Dstart.home=%START_HOME% com.runqian.starter.StartUtil com.runqian.starter.EmbedeDataBaseProcessor start

start “runqianreport webserver” %EXECJAVA% -cp %RunqianCLASSPATH% -Dstart.home=%START_HOME% com.runqian.starter.StartUtil com.runqian.starter.EmbededTomcatProcessor start demo

保存后，当启动设计器的时候demo应用也会随之启动的。
